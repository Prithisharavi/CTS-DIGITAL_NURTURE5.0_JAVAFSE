# Exercise 10 - Hibernate Specific Features

## Objective
Implement Hibernate-specific features to improve application performance and capabilities.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database
- Maven

## Hibernate Features Implemented
- @DynamicInsert
- @DynamicUpdate
- Hibernate Dialect Configuration
- Hibernate Batch Processing using saveAll()

## REST APIs Tested

POST /departments

POST /employees

POST /employees/batch

GET /employees

## Result
Successfully implemented Hibernate-specific annotations, configured Hibernate properties, and performed batch insertion of employees.
