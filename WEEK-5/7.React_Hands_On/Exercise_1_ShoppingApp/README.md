# Exercise 1 - Shopping App using Props

## Objective
Create a React application that demonstrates the use of Props by passing shopping cart data from a parent component to a child component.

## Tools Used
- React
- JavaScript
- Visual Studio Code

## Features
- Uses React Props to pass data.
- Displays shopping items in a table.
- Parent component passes an array of items.
- Child component renders the received data.

## Output
Displays a table containing item names and prices using React Props.
