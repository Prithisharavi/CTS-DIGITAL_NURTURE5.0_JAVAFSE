# Exercise 1 - My First React Application

## Objective
Create a simple React application that displays:

**Welcome to the first session of React**

## Technologies Used
- React
- JavaScript
- Node.js
- npm
- Visual Studio Code

## Steps
1. Created a React application using `create-react-app`.
2. Modified `App.js`.
3. Ran the application using `npm start`.
4. Verified the output in the browser.

## Output

Welcome to the first session of React
