# Exercise 5 - Styling React Components

## Objective
Style an existing React application using CSS Modules and inline CSS.

## Tools Used
- React
- CSS Modules
- JavaScript
- Visual Studio Code

## Features
- Styled cohort cards using CSS Modules.
- Applied inline color styling based on cohort status.
- Green color for Ongoing cohorts.
- Blue color for Scheduled cohorts.

## Output
Displays styled cohort details with different colors based on status.
