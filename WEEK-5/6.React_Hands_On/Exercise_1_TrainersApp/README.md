# Exercise 6 - React Router

## Objective
Create a React application that demonstrates routing using React Router.

## Tools Used
- React
- React Router DOM
- JavaScript
- Visual Studio Code

## Features
- Navigation using React Router.
- Home page for trainers.
- Trainers list with clickable links.
- Trainer details displayed using URL parameters.
- Dynamic routing with useParams().

## Output
The application displays a home page, trainers list, and trainer details through React Router navigation.
