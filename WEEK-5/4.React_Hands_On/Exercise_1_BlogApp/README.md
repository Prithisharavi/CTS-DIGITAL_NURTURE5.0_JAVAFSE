# Exercise 1 - Blog App

## Objective
Create a React application that fetches and displays blog posts using React lifecycle methods.

## Tools Used
- React
- Axios
- JavaScript
- Visual Studio Code

## Features
- Fetches blog posts from REST API.
- Uses componentDidMount() lifecycle method.
- Displays post title and body.
- Handles API errors using componentDidCatch().

## API Used
https://jsonplaceholder.typicode.com/posts

## Output
The application displays a list of blog posts retrieved from the API.
