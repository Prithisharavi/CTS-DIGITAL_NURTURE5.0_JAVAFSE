# Exercise 2 - Score Calculator App

## Objective
Create a React application using a functional component to calculate and display a student's score percentage.

## Features
- Accepts student details using props.
- Calculates the percentage score.
- Displays:
  - Student Name
  - School Name
  - Total Marks
  - Percentage Score
- Applies CSS styling using an external stylesheet.

## Technologies Used
- React JS
- JavaScript (ES6)
- CSS
- Node.js
- npm
- Visual Studio Code
