# Exercise 8 - Counter App (React State)

## Objective
Create a React application that demonstrates the use of the React State object.

## Description
This application contains a `CountPeople` component with two buttons:

- **Login** – Increases the number of people entered.
- **Exit** – Increases the number of people exited.

The application uses React State to update and display the counts dynamically.

## Technologies Used
- React JS
- JavaScript
- HTML
- CSS

## Output
- Displays the number of people entered and exited.
- Counts are updated when the Login and Exit buttons are clicked.
