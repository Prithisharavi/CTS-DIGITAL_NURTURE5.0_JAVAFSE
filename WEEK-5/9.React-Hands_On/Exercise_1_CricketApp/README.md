# Exercise 9 - Cricket App (ES6 Features)

## Objective
Create a React application to demonstrate the ES6 features such as map(), arrow functions, destructuring, and the spread operator.

## Description
This application displays a list of cricket players and demonstrates various ES6 concepts.

### Features
- Display all players using the `map()` method.
- Display players having scores less than 70 using arrow functions.
- Display odd and even Indian team players using array destructuring.
- Merge T20 and Ranji Trophy players using the spread operator.

## Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

## ES6 Features Used
- map()
- Arrow Functions
- Array Destructuring
- Spread Operator
- Conditional Rendering
