# Exercise 1 - Hello World RESTful Web Service

## Objective
Develop a RESTful Web Service using Spring Boot that returns the message **"Hello World!!"**.

## Technologies Used
- Java 17
- Spring Boot 3
- Spring Web
- Maven
- SLF4J Logging

## REST Endpoint

### GET /hello

**URL**

http://localhost:8083/hello

**Response**

Hello World!!

## Files Used
- SpringLearnApplication.java
- HelloController.java
- application.properties

## Output
Successfully created a RESTful Web Service using `@RestController` and `@GetMapping` that returns **Hello World!!**.
