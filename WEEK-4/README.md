# Microservices with Spring Boot 3 and Spring Cloud

This repository contains hands-on exercises on Microservices using Spring Boot 3 and Spring Cloud.

## Technologies Used
- Java 17
- Spring Boot 3
- Spring Cloud
- Maven
- REST API
- Spring Cloud Gateway
- Eureka Discovery Server
- Spring Cloud Config Server

## Exercises

### 1. Microservices using Spring Boot 3
- Account Service
- Loan Service
- REST APIs
- Independent Microservices

### 2. Microservices with API Gateway
#### Exercise 1
- Account Service
- Loan Service
- Eureka Discovery Server
- API Gateway
- Global Logging Filter

#### Exercise 2
- Product Service
- Inventory Service
- Eureka Discovery Server
- Config Server
- API Gateway
- Centralized Configuration
