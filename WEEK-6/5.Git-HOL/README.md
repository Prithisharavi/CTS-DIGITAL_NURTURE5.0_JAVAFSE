# Git HOL 5 – Clean Up and Push to Remote Git

## Objective
To clean up the local repository and push the latest changes to the remote Git repository.

## Prerequisites
- Git installed
- GitHub repository configured
- Local repository connected to remote repository

## Commands Executed

```bash
git status
git branch -a
git pull origin main
git push origin main
```

## Steps Performed

1. Verified the repository was in a clean state using `git status`.
2. Listed all available local and remote branches.
3. Pulled the latest changes from the remote repository.
4. Pushed the local repository to the remote GitHub repository.
5. Verified that the local and remote repositories were synchronized.

## Output

- Working tree was clean.
- Branch list displayed successfully.
- Repository was already up to date.
- Push operation completed successfully with no pending changes.

## Result

Successfully verified the repository status and synchronized the local repository with the remote GitHub repository.
